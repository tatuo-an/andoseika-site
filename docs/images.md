# 画像アセット管理ガイド

このドキュメントでは、Webサイトで使用する画像の配置場所、ファイル名、推奨サイズについてまとめています。
画像を差し替える際は、このガイドに従ってファイルを上書き保存してください。

## 1. 基本ルール
*   **ファイル形式**: 写真は **`.jpg`** (または `.jpeg`) を推奨します。イラストや透過が必要なロゴは `.png` を使用します。
*   **ファイル名**: コード内で指定されているため、**ファイル名は変更しないでください**。
*   **更新方法**: 新しい画像を同じファイル名で保存し、既存のファイルを上書きしてください。

## 2. 画像一覧と配置場所

プロジェクトの `public/images/` フォルダ配下に、以下の構成で配置してください。

### トップページ (Top Page)
| 箇所 | パス (ファイル名) | 推奨サイズ | 比率 | 備考 |
| :--- | :--- | :--- | :--- | :--- |
| ヒーロー画像 (背景) | `/public/images/hero/hero.jpg` | 1920 x 1080 px | 16:9 | PC全画面用。容量は300KB以下推奨。 |
| Aboutセクション | `/public/images/about/about.jpg` | 800 x 600 px | 4:3 | 農園の雰囲気が伝わる写真。 |

### 商品一覧ページ (Products)
商品は正方形（1:1）または少し横長（4:3）で統一すると綺麗に見えます。

| 商品名 | パス (ファイル名) | 推奨サイズ | 備考 |
| :--- | :--- | :--- | :--- |
| 白ネギ | `/public/images/products/negi.jpg` | 800 x 800 px | |

## 3. 注意事項
*   **キャッシュについて**: 画像を差し替えてもブラウザ上で変わらない場合、キャッシュが残っている可能性があります。ブラウザをスーパーリロード（Shift + Cmd + R）するか、シークレットウィンドウで確認してください。
*   **容量**: スマホでの表示速度に影響するため、1枚あたり **500KB以下**（できれば200KB前後）に圧縮してから配置することをお勧めします。
